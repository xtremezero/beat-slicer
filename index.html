How to Play
1. Load an audio track and click START.
2. Slice through blocks to the beat using mouse drag or touch swipe (multi-touch supported!).
3. Dot blocks can be sliced in any direction.
Game Modes
NORMAL: Swipe in the arrow’s direction to hit arrow blocks. You can also press the matching Arrow / WASD key to hit the closest block directly.
OMNI-DIRECTIONAL: All blocks become dots — slice in any direction with no key required. Great for relaxing!
CHALLENGE: You must hold the correct Arrow / WASD key while swiping through arrow blocks. Keyboard-only hits are disabled. The ultimate test!
